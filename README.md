## Gestor de tareas

> [!NOTE]
> Al clonar el repositorio, ejecutar `npm install` para instalar las dependencias.
> 
> Para ejecutar la aplicación, ejecutar `npm run start`.

Aplicación de notas creada con Electron que permite crear, ver y eliminar notas.

## Características

- Crear notas
- Ver notas
- Eliminar notas
- Guardar notas en una base de datos SQLite
